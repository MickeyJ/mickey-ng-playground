## Angular-Playground

__Install errthing__
```bash
  
$ npm install
  
```

__Webpack Development Server__ [Port:8080](http://localhost:8080/)
```bash
  
$ npm run dev
  
```

__Test Minified Angular__ [Port:8080](http://localhost:8080/)
```bash
  
$ npm run stage
  
```

__Bundle and run with Express__ [Port:3000](http://localhost:3000/)
```bash
  
$ npm run build
  
```
